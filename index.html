Computer Network Project Report

  • Project Task:
    • How the site accesses its networking capabilities:
        • Vercel's edge network lets the user store content close to their customers and compute in the region close to their data by reducing latency and improving end-user performance.
    • How the hosting service works
        • Vercel is a cloud platform that helps developers build and deploy web projects. It offers a global CDN, and automatically gives the site an HTTPS interface. It also uses
        smart routing to direct requests to the closest data center, which can help speed up website loading.
    • Describe how the site and host connect
        • They connect to their host by integrating users with a user's Git repository(GitHub), where the website code is stored. When changes are pushed to the repository Vercel automatically builds and deploys the site to its server network, making the website accessible through a unique URL.
